# Go 学习背景

这个目录用于保存学习 Go 的背景资料、研究结论和项目设计草稿。它不是最终展示站点的唯一来源，而是更偏向“思考和决策记录”的工作区。

## 当前目标

用项目驱动的方式学习 Go：

1. 先建立 Go 的工程心智模型，而不是孤立背语法。
2. 以一个 Hacker News 摘要推送 Agent 作为贯穿练习。
3. 把学习过程沉淀成可复用的 Markdown 知识库。
4. 用 AI 辅助生成、重构、review 和补测试，但核心判断保持由开发者掌握。

## 目录建议

- `research-summary.md`：研究报告的简体中文整理。
- `learning-roadmap.md`：阶段化学习路径。
- `js-to-go.md`：从 JavaScript / Node.js 到 Go 的心智迁移。
- `hn-agent.md`：Hacker News 摘要推送 Agent 的项目方案。
