# 00 从零开始

## 本步目标

明确项目到底要做什么、第一版不做什么，以及学习如何和项目交付绑定。

## 项目一句话

用 Go 构建一个 Hacker News 摘要推送 Agent：定时拉取热门 story，生成简体中文摘要，并提供查询或推送能力。

## 第一版范围

第一版只追求闭环：

- 拉取 HN `topstories`。
- 拉取 story item 详情。
- 生成本地 digest。
- 暴露一个查询最新 digest 的 API。
- 有基础测试。

第一版暂不追求：

- 完整 Web dashboard。
- 多用户系统。
- 复杂权限。
- 多渠道推送。
- 高级推荐算法。

## 学习绑定

| 学习阶段 | 实战对应 |
| --- | --- |
| 启动期 | 创建 Go module 和 CLI |
| 网络期 | 调用 HN API |
| 并发期 | 批量抓取 story |
| 服务期 | 启动 HTTP API |
| AI 整合期 | 生成结构化摘要 |
| 验证与部署期 | 测试、CI、Docker |

## 完成标准

- 能说清楚 MVP 的输入、处理和输出。
- 能说清楚第一版不做什么。
- 能确定下一步是创建 Go 项目骨架。
